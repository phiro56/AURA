# Changelog

All notable changes to AURA will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-31

### Added
- **SOMA extension** — Soft skills and capabilities layer
- **ClawHub skill** — Interactive `/aura` command for easy setup
- **TypeScript implementation** — `aura-loader` npm package
- **6 archetype profiles** — Analyst, Creative, Mentor, Executive, Diplomat, Scientist
- **JSON Schema** — Full validation for AURA.yaml files

### Changed
- Improved documentation with installation guides
- Better examples in SPEC.md

## [1.0.0] - 2026-01-27

### Added
- Initial AURA specification
- HEXACO-based personality traits
- Style attributes (formality, verbosity, humor, assertiveness, autonomy)
- Boundaries (max_adulation, always_correct_errors, flag_uncertainty)
- Basic profiles (balanced, direct, warm, clinical)
- Science documentation with HEXACO references
